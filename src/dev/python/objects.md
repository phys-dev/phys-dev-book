## –û–±—ä–µ–∫—Ç—ã –≤ –ø–∞–º—è—Ç–∏

–í Python **–≤—Å—ë —è–≤–ª—è–µ—Ç—Å—è –æ–±—ä–µ–∫—Ç–æ–º**. –ö–∞–∂–¥—ã–π –æ–±—ä–µ–∫—Ç –∏–º–µ–µ—Ç:

### –°—Ä–∞–≤–Ω–µ–Ω–∏–µ: `is` vs `==`

- `is` –ø—Ä–æ–≤–µ—Ä—è–µ—Ç, —Å—Å—ã–ª–∞—é—Ç—Å—è –ª–∏ –¥–≤–µ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–µ –Ω–∞ **–æ–¥–∏–Ω –∏ —Ç–æ—Ç –∂–µ –æ–±—ä–µ–∫—Ç**.
- `==` –ø—Ä–æ–≤–µ—Ä—è–µ—Ç, **—Ä–∞–≤–Ω—ã –ª–∏ –∑–Ω–∞—á–µ–Ω–∏—è** –æ–±—ä–µ–∫—Ç–æ–≤.

–ü—Ä–∏–º–µ—Ä:
```python

```

> üí° **–ó–æ–ª–æ—Ç–æ–µ –ø—Ä–∞–≤–∏–ª–æ**: –≤—Å–µ–≥–¥–∞ –∏—Å–ø–æ–ª—å–∑—É–π—Ç–µ `==` –¥–ª—è —Å—Ä–∞–≤–Ω–µ–Ω–∏—è –∑–Ω–∞—á–µ–Ω–∏–π, –∞ `is` ‚Äî —Ç–æ–ª—å–∫–æ –¥–ª—è –ø—Ä–æ–≤–µ—Ä–∫–∏ –Ω–∞ `None`.
